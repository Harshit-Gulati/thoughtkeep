# 🧠 ThoughtKeep

**ThoughtKeep** is a modern web application that acts as your **second brain** for storing and managing all your important links. Built with the **MERN Stack** (MongoDB, Express.js, React.js, Node.js) and **TypeScript**, ThoughtKeep ensures seamless organization and accessibility for your digital resources.

## 🌐 Live Demo  
You can access the live app here: [ThoughtKeep](https://thoughtkeep.vercel.app)

---

## 🚀 Features

- **Link Management**: Easily add or delete links.
- **Categorization**: Organize links into different categories for better accessibility.
- **Shareable Brain**: Share your organized brain with others, enabling shared access to important links.
- **User Authentication**: Securely sign up, log in, and manage your account using **JWT-based authentication**.
- **Responsive Design**: Enjoy a seamless experience across devices.

---

## 🛠️ Tech Stack

- **Frontend**: React.js with TypeScript and Tailwind CSS for styling.
- **Backend**: Express.js and Node.js for the server-side logic.
- **Database**: MongoDB for storing user data and links.
